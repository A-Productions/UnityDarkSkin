## Usage
1. Compile and copy UnityDarkSkin.exe to folder with Unity.exe
2. Run UnityDarkSkin.exe as administrator

## Supported versions
* 5.0 to 2018.2 (tested on 5.3, 5.4, 2017.2, 2018.2)
* 2018.3 to 2019.1 (tested on 2018.3)
